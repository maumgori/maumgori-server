#마음고리 서버 프로그램

이 프로그램을 실행하려면 Node.JS가 필요합니다.

###사용된 프로그램 및 라이브러리.
- node.js 0.12.0
- nodemon 1.3.7
- express 4.12.2
- body-parser 1.12.0
- jade 1.9.2
- yamljs 0.2.1
- stylus 0.50.0
- bower 1.3.12
- bootstrap 3.3.2
- toastr 2.1.0
- angular.js 1.3.14

### 프로그램 설치.
Node.js가 먼저 설치 되어 있어야 합니다.

프로젝트를 Clone 받거나 다운로드 해서 압축 푼 후 홈 디렉토리에서 다음 명렁어 실행합니다.
```
npm install
node_modules/.bin/bower install
```

마음고리 쉘 실행.
```
./maum
```

로컬 호스트에 접속.
http://localhost:3000
